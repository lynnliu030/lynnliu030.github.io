Creating Personal Blog Using Github + Hexo

Posted on 2018-06-05

A brief guideline of creating a personal website. This guideline includes the solutions to some common errors.

Intro

This is my first blog :)

In this article, I will record my process of creating a personal blog
using Github and Hexo. This approach could be used to create any websites, not just blog. I will also discuss about some problems I met with and the solutions I used during this process, which might be helpful for somebody who visits this website. Hope you enjoy the charm of creation.

Github

Github Pages is a static site hosting service designed to host your personal, organization, or project pages directly from a Github repository.

If you don’t have a Github account, create one here!

Hexo

What is Hexo?

Hexo is a fast, simple and powerful blog framework; it’s also a static site generator. Click hexo website to get more information.

Webstie Optimization

1. Add Favicon to Hexo Blog

It depends on theme you’re using. Check theme documentation and the source.

Here, I would like to introduce the way to change Favicon under the nexT theme. (note: Favicon means the website icon)

The size of favicon could be 16x16, or 32x32. You could download the icon from any websites you like. There’re many websites that provide cute icons, such as fasticon, easyicon and so on.

Notice, the picture must be the right size. Or you could use this Favicon Generator to convert your images into an icon favicon.ico.

If you didn’t use the website above, please remember to change the name of your image to “favicon.ico”.
