Creating Personal Blog Using Github + Hexo

Posted on 2018-06-05

Intro

This is my first blog :)

In this article, I will record my process of creating a personal blog using Github and Hexo. This approach could be used to create any websites, not just blog. I’ll also discuss about some problems I met with and the solutions I used during this process, which might be helpful for somebody who visits this website. Hope you enjoy the charm of creation.

Outline

  • Github
  • Hexo (Node.js, Git)
  • Website Creation
  • Markdown

Github

Github Pages is a static site hosting service designed to host your personal, organization, or project pages directly from a Github repository.

  • If you don’t have a Github account, create one here! We need it to build our website.

Hexo

What is Hexo?

Hexo is a fast, simple and powerful blog framework; it’s also a static site generator. Click hexo website to get more information.
